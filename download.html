CrawlScript 下载

下载

CrawlScript在github上发布，可用下面三种方式下载：

1.进入CrawlScript的github主页,右下方的Download Zip按钮下载.

2.直接点击下载CrawlScript.

3.如果电脑装有git命令,可用git命令下载:
  git clone https://github.com/CrawlScript/CrawlScript
